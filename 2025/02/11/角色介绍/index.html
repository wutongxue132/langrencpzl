角色介绍

发表于 2025-02-11 更新于 2025-02-14 分类于 介绍文档 本文字数： 1.2k 阅读时长 ≈ 4 分钟

狼人阵营

普通狼人

  • **介绍：**每晚可以与其他狼人一起选择一名玩家击杀。狼人必须团结合作，共同选择一个目标，是狼队的主要输出手。
  • **技能：**每晚和其他狼人一起睁眼，选择一名玩家击杀。

狼美人

  • **介绍：**每晚可以魅惑一名玩家，如果狼美人被票出或死亡，被魅惑的玩家也会跟随死亡。是狼队强大的战略角色，可以通过魅惑来威胁或策反玩家。
  • **技能：**每晚可以魅惑一名玩家，如果狼美人被票出或死亡，被魅惑的玩家也会跟随死亡。

白狼王

  • **介绍：**可以在白天自爆并带走一名玩家。这是一个极具牺牲精神的角色，在关键时刻可以通过自爆来扭转局势，带走对狼人威胁最大的玩家。
  • **技能：**白天可以自爆并带走一名玩家，自爆前需向所有玩家亮明身份。

隐狼

  • **介绍：**被预言家查验时显示为好人阵营。当其他狼人全部死亡时，可以在夜晚单独击杀一名玩家。是狼队的隐藏战力，可以迷惑预言家和好人阵营。
  • **技能：**被预言家查验时显示为好人阵营。当其他狼人全部死亡时，可以在夜晚单独击杀一名玩家。

狼王

  • **介绍：**被票出局时可以开枪带走一名玩家。作为狼队的核心角色，即使被投票出局也能发挥最后的价值，是狼人阵营的重要战术角色。
  • **技能：**被投票出局时可以开枪带走一名玩家，开枪前需向所有玩家亮明身份。

恶灵骑士

  • **介绍：**免疫女巫的毒药，被预言家查验时会导致预言家出局，被骑士决斗时骑士出局而恶灵骑士存活。是一个极其强大的特殊狼人，克制多个神职。
  • **技能：**1. 免疫女巫的毒药 2. 被预言家查验时会导致预言家出局

好人阵营

预言家

  • **介绍：**每晚可以查验一名玩家的真实身份，是否属于狼人阵营。是好人阵营的情报核心，但查验恶灵骑士时会导致预言家出局，需要谨慎行动。
  • **技能：**每晚可以查验一名玩家的真实身份，但查验恶灵骑士时会导致预言家出局。

女巫

  • **介绍：**拥有一瓶解药和一瓶毒药。每晚只能使用一瓶药。解药可以救活被狼人杀害的玩家，毒药可以毒死一名玩家。是好人阵营的强力辅助，需要合理使用两瓶药。
  • **技能：**1. 拥有一瓶解药和一瓶毒药 2. 每晚只能使用一瓶药 3. 解药可以救活被狼人杀害的玩家 4. 毒药可以毒死一名玩家

猎人

  • **介绍：**被狼人杀害或被投票出局时可以开枪带走一名玩家。但被女巫毒死时不能开枪。是好人阵营的重要反击角色，死亡时的开枪能力可以反杀狼人。
  • **技能：**被狼人杀害或被投票出局时可以开枪带走一名玩家，但被女巫毒死时不能开枪。

好人阵营

  • **介绍：**每晚可以守护一名玩家免受狼人伤害，但不能连续两晚守护同一个人。是好人阵营的防御型角色，合理的守护可以保护重要神职。
  • **技能：**每晚可以守护一名玩家免受狼人伤害，但不能连续两晚守护同一个人。

骑士

  • **介绍：**可以在白天发起决斗。如果目标是狼人，则狼人出局；如果目标是好人，则骑士出局；如果目标是恶灵骑士，则骑士出局。是一个需要慎重使用技能的验人角色。
  • **技能：**可以在白天发起决斗：1. 目标是狼人，则狼人出局 2. 目标是好人，则骑士出局 3. 目标是恶灵骑士，则恶灵骑士出局

白痴

  • **介绍：**被投票时不会出局，永远不会因为投票而死亡。是一个特殊的保命角色，可以在关键时刻承担票型压力，帮助好人阵营获取更多信息。
  • **技能：**被投票时不会出局，永远不会因为投票而死亡，但仍会死于狼人袭击、女巫毒药等其他方式。
